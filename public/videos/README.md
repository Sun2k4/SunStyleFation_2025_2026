# Videos Directory

Place your videos here. They will be accessible at `/videos/filename.ext`

## Example usage in code:
```jsx
<video autoPlay muted loop>
  <source src="/videos/hero-video.mp4" type="video/mp4" />
</video>
```

## Recommended video formats:
- `.mp4` - Best compatibility (H.264 codec)
- `.webm` - Better compression, modern browsers

## Tips for Hero videos:
- Keep duration: 5-10 seconds
- Keep size: < 2MB
- Resolution: 1920x1080 or 1280x720
- Remove audio track to reduce size
- Use tools like HandBrake to compress
